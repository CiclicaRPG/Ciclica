Processo Opirítico: Ponto de Saída

Permite que o invocador transporte objetos consigo para outro local.

Processo Opirítico: Ponto de Saída

[Fechar]
Com este processo é possível transportar objetos do invocador ou de um ponto de entrada para o ponto de saída.

Requerimento: poder 2 e Invocador 1

Via: Instantâneo

Velocidade Extra: -4 + Invocador

Efeito: O invocador consome uma carga para criar um objeto de seu ícor que existe por 20*(cargas) horas. Enquanto o objeto persistir, o invocador pode consumir cargas para transportar objetos a partir de si, para espaços adjacentes ao ponto de saída.

  • 1 carga: objetos pequenos de ate 50 cm que exigem apenas uma mão livre para utilizar.
  • 2 cargas: objetos grandes ou indivíduos de até 2 metros.
  • 3 cargas: até dois grandes objetos ou indivíduos de até 3 metros.
Processo Opirítico: Ponto de Entrada

Permite o invocador trazer objetos adjacentes ao ponto de saída para o ponto de entrada ou para si.

Processo Opirítico: Ponto de Entrada

[Fechar]
Com este processo é possível transportar objetos ponto de saída para o invocador ou para o ponto de entrada.

Requerimento: poder 2 e Invocador 2

Via: Instantâneo

Velocidade Extra: -4 + Invocador

Efeito: O invocador consome uma carga para criar um objeto de seu ícor que existe por 20*(cargas) horas. Ao invés de criar um objeto, o próprio invocador pode servir como ponto de entrada. Enquanto o objeto persistir, o invocador pode consumir cargas para transportar objetos adjacentes ao ponto de saida, para espaços adjacentes ao ponto de entrada.

  • 1 carga: objetos pequenos de ate 50 cm que exigem apenas uma mão livre para utilizar.
  • 2 cargas: objetos grandes ou indivíduos de até 2 metros.
  • 3 cargas: até dois grandes objetos ou indivíduos de até 3 metros.
